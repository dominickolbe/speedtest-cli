```bash
npx @dominickolbe/speedtest-cli
```

## Installation

via [npm](https://www.npmjs.com/package/@dominickolbe/speedtest-cli) :package:
```bash
npm install --global @dominickolbe/speedtest-cli
```

## Usage
```bash
$ speedtest-cli
```
